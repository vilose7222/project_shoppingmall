JSP 로 MVC 패턴을 활용한 쇼핑몰 프로젝트

## 개발 인원

## 맡은 기능

1.회원가입 , 로그인 , 로그아웃 등 회원관리

2.DB와 연동한 제품 리스트, 상세페이지
